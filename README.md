# Sri Kavin's Tailors

A modern tailor catalogue web application built using the MERN stack.
Customers can explore blouse and chudi designs, search by model ID, and directly share designs through WhatsApp.

## 🚀 Features

* Design catalogue display
* Search by Model ID
* Category filtering
* WhatsApp sharing
* Product view page
* Admin upload panel
* Edit & delete designs
* Cloudinary image upload
* Responsive UI
* Modern glassmorphism design

## 🛠️ Technologies Used

### Frontend

* ReactJS
* React Router DOM
* Axios
* React Icons
* CSS3

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Multer
* Cloudinary

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Nandhini-154/Sri-Kavin-Tailors.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

## 🌐 Deployment

* Frontend: Netlify
* Backend: Render
* Database: MongoDB Atlas
* Image Storage: Cloudinary
